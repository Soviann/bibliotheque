## Stack technique

| Composant | Version | Usage |
|-----------|---------|-------|
| PHP | 8.3+ | Backend |
| Symfony | 7.4 | Framework |
| API Platform | 4 | API REST (JSON-LD) |
| MariaDB | 10.11+ | Base de données |
| React | 19 | Frontend |
| TypeScript | 5.9 | Typage |
| Vite | 7 | Bundler |
| TanStack Query | 5 | State management serveur |
| Tailwind CSS | 4 | Styling |
